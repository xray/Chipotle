# Chipotle (C#/.NET)
C# Version 7.2 | .NET Version 2.2.0
  
## Requirements
* .NET SDK 2.2  
  
## Usage
* In terminal from the project folder navigate to the `src` folder.
* Run `dotnet run`
* If you have access to the acceptance test suite you can run them against this server
  
## Tests
How to run the tests:  
* Run `dotnet test`
